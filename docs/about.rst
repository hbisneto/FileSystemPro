About FileSystemPro
===================

FileSystemPro is a Python library designed to simplify file and directory operations across multiple operating systems. It provides a user-friendly interface for tasks such as creating, deleting, and managing files and directories, with robust tools for compression, system monitoring, and terminal styling.

**Key Features**:
- Cross-platform compatibility (Windows, macOS, Linux)
- Simple and intuitive API
- Robust error handling
- Comprehensive file and directory management
- Support for compression (tar and zip formats)
- System monitoring and device information retrieval
- Colored terminal output with ANSI escape sequences
